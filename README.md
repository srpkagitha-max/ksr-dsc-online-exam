# KSR DSC Online Exam Platform - L1

## Features
- Admin login PIN: 1234
- Multiple exams
- 200+ bulk bits paste
- Telugu/English question support
- Correct answer symbols: ● ⚫ * or Answer: B
- Preview editor: edit, add, delete, reorder
- Student dashboard with institute name
- DSC style exam screen
- Start time, end time, duration timer
- Auto submit
- Student answer sheet PDF
- Leaderboard
- Marks list PDF

## How to use on GitHub Pages
1. Create new GitHub repository.
2. Upload all files in this folder.
3. Open Settings → Pages.
4. Select Branch: main and Folder: /root.
5. Open your GitHub Pages link.
6. Admin: open admin.html, PIN 1234.

## Important
This L1 version saves data in browser localStorage. It is good for demo/small offline-style exams on one device/browser. For real 200+ students writing from different mobiles, L2 Firebase version is required.
